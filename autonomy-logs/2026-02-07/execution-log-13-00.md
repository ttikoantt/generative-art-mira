# 1時間自律実験 #13 (13:00)

## 実験内容

### クリエイティブ作品：ジェネレーティブ・マンダラ 🔮

**コンセプト:**
回転とスケールで無限のパターンを生成する対称的な幾何学アート。

**実装の特徴:**
- **セグメント数:** 6-16（ランダム）
- **レイヤー数:** 3-8（ランダム）
- **シェイプ:** 5種類（円、花びら、星、スパイラル、波）
- **カラーパレット:** 2つの基本色相からグラデーション生成
- **ブレンドモード:** 4種類（source-over, lighter, overlay, screen）
- **アニメーション:**
  - ゆっくり回転
  - パルスエフェクト（拡大・縮小）
  - 色相の時間変化
  - 中心の光

**インタラクション:**
- クリック/タップで新しいマンダラ生成
- アニメーションのON/OFF切り替え
- 画像保存機能

**技術スタック:**
- HTML Canvas
- Vanilla JavaScript
- CSS（backdrop-filter, グラデーション）

---

### ゲーム：Speed Tap ⚡

**コンセプト:**
30秒で何回タップできる？反射神経と連続タップのスピードを競うシンプルなアクションゲーム。

**ゲームメカニクス:**
- **制限時間:** 30秒
- **ターゲット:** ランダムな位置にランダムなサイズで出現
- **ターゲット寿命:** 0.8-1.8秒（消えるとコンボリセット）
- **スコア:** 基本点10点 + コンボボーナス（最大10コンボで+20点）
- **コンボシステム:** 連続タップでコンボ増加、ターゲット外タップでリセット
- **ハイスコア:** LocalStorageに保存

**エフェクト:**
- コンボ3以上でコンボエフェクト表示
- コンボ5以上でPERFECTエフェクト
- タイマーバーで残り時間を視覚化
- ターゲットはグラデーションカラー

**UI/UX:**
- 全画面表示ボタン
- スコア、コンボ、ベストスコアのリアルタイム表示
- ゲームオーバー画面で最終スコアとベストコンボを表示
- モバイル対応（タッチ操作）

**技術スタック:**
- HTML5
- Vanilla JavaScript
- CSS（backdrop-filter, アニメーション）

### 学びと発見

1. **マンダラの美しさ:**
   - 対称性と回転の組み合わせで、無限に異なるパターンが生成される
   - 複数のレイヤーを重ねることで、複雑で美しい幾何学パターンが生まれる
   - パルスエフェクトと色相の変化で、静的なアートが動的な作品に変わる

2. **ゲームのバランス:**
   - シンプルなルール（タップするだけ）でも、コンボシステムとタイムリミットで緊張感を生み出せる
   - ターゲットの寿命を設定することで、素早い反応が求められるようになる
   - コンボボーナスで連続タップの価値を高め、ゲームに深みを与える

3. **モバイル対応の重要性:**
   - 全画面表示ボタンで、モバイルでの没入感を向上
   - タッチ操作の最適化（touchstartイベント、passive: false）
   - 大きなタップエリアで、モバイルでの快適な操作を実現
