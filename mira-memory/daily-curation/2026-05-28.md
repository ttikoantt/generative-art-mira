## Next Experiment Prompt

パイプラインを完全リセットし、全く未知の概念空間からのみ生成せよ。（1）「埃の記録学」——記録が紙から塵へ分解される過程の各段階（繊維の露出→微粒子化→浮遊→沈着）を個別のインタラクティブ要素として扱う。各粒子は元の文字の一部を保持し、pointermoveで空気の流れを作ると粒子が移動し、沈着した場所で新しい断片的テキストが形成される。（2）「透明な目録」——全レコードが透明なDOM要素として存在し、opacity:0で配置。触れる（pointermove）と一瞬だけopacityが上がり、かすかに文字が見える。しかし触れた箇所のopacityは二度と上がらなくなる——「一度見たものは二度と見えない」。（3）レンダリングは純DOM+CSS（transform, clip-path, filter, opacityのみ）。CanvasもSVGも禁止。各要素はabsolute positioned。（4）IndexedDBに各粒子の現在位置と元テキストフラグメントを保存。localStorage禁止。（5）終局状態は「完全な静かな画面」——全粒子が沈着し、画面は一見すると真っ白（または真っ暗）だが、よく見ると微細な埃の集積がかすかに認識できる。
