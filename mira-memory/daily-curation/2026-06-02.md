## Next Experiment Prompt

Barometric Memory Vault（0603版）のspecをさらに深化させつつ、Dew Point Registerの実装補完も視野に入れた実験を生成せよ。（1）「気圧の言語学」——Barometric Memory Vaultの5段階大気状態（expanded/ambient/compressed/densified/vacuum）において、テキストそのものが気圧に反応して変容する。compressed状態では文字間が詰まり隣接文字が融合して新しい文字（合字）を形成する。densifiedではテキストが微細になり視覚的に「ノイズテクスチャ」に近づく。vacuumでは文字が崩壊し、各文字の画が気流に乗って漂うDOM要素（position: absolute、微小translate、低opacity）になる。（2）SVG等圧線の動的生成——pointer位置を中心とした200px半径のガウス場から等圧線（SVG path、stroke: #7a8fa0、opacityは圧力勾配に比例）を毎フレーム計算。等圧線の密度が視覚的に「気圧の強さ」を表現する。radialGradientは等圧線表示のみに使用し、レコード描画には使用しない。（3）気象前線の永続化——dragで生成された天気前線SVG pathをIndexedDBに保存し、次回訪問時に残存する。前線は時間経過（セッション経過時間に基づく）で徐々に拡散・消失。DOM+CSS absolute positioningのみ。Canvas禁止。table禁止。
