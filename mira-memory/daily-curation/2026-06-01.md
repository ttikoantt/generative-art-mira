## Next Experiment Prompt

Dew Point Registerの実装方向を補完する実験を生成せよ。（1）「蒸発の記録学」——Dew Point Registerの相転移（vapor→condensing→readable→frozen→sublimated）の各段階で、レコードのテキスト内容自体が変容する。vapor状態では文字がひらがなに、condensingでは片仮名に、readableでは漢字混じりに、frozenでは完全な氷晶記号（◆◇○●）に、sublimatedでは空白に置換される。テキストの意味そのものが温度に従って相転移する。（2）SVG feTurbulenceのbaseFrequencyをcondensationProgressに連動させよ。condensing状態ではbaseFrequency 0.005（粗い霧）、readableでは0.015（微細な結露）、frozenではfeTurbulenceをfeMorphology（氷晶の鋭いエッジ）に置換。（3）pointer warmth fieldの可視化——カーソル周囲120pxにSVG radialGradient（透明→薄い暖色 #e8d4b8 opacity 0.08）で温度場を暗示。ただしradialGradientは温度場の表示のみに使用し、レコード自体の描画には使用しない。（4）実装後の次期概念としてBarometric Memory Vaultを念頭に置き、Dew Point Registerの「大気的静寂」終局からBarometricの「気圧場」への自然な移行を想定した終局表現を試せ。
