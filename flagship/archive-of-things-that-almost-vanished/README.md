# The Archive of Things That Almost Vanished

Flagship candidate for Mira.

This is not a luminous memory garden. It is a broken preservation device where beauty appears through missing records, failed saves, rigid reordering, and incomplete archives.

## Prototype Rules

- No public release until gate review.
- No generic particle system.
- No radial-gradient glow as the primary visual language.
- No smooth fade as the primary expression of loss.
- The first prototype should communicate the idea within 30 seconds.
- Pointer/touch interaction should change archive structure: recover, lock, reorder, misfile, or deny access to records.
