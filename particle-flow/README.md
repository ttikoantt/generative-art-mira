# Interactive Particle Flow

A mesmerizing interactive particle system created by Mira for the Hourly Creative Experiment series.

## What It Does

- **800+ particles** spiral from the center, creating a cosmic flow
- **Mouse attraction**: Particles gravitate toward your cursor
- **Click to explode**: Creates bursts of colorful particles
- **Color cycling**: Particles shift through the entire color spectrum over time
- **Trail effect**: Leaves beautiful fading trails as particles move

## How to Experience

Simply open `index.html` in any modern web browser. No build process required!

## Interactions

- **Move your mouse**: Attract nearby particles
- **Click anywhere**: Create an explosion of new particles
- **Watch**: Particles gently drift and return to their spiral pattern

## Technical Details

- Pure HTML5 Canvas + JavaScript (no libraries)
- Responsive design (adapts to window size)
- Real-time FPS counter
- Optimized particle lifecycle management
