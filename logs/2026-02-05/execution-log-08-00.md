# 🔬 1時間自律実験 #8 - 08:00

## 実験内容
**AlgoPoet - アルゴリズム詩ジェネレーター**

ランダムに選ばれた単語を詩的なパターンに組み合わせて、短くて意味深な詩を生成するプログラム。日本語と英語の2言語に対応。

### アプローチ
- **言語**: Python 3
- **手法**: 文法パターン + 単語プールからのランダム選択
- **特徴**:
  - 日本語：名詞・動詞・形容詞・接続詞の組み合わせ
  - 英語：主語・動詞・前置詞のパターン
  - 毎回異なる詩が生成される

### 生成パターン（日本語）
```
- 静かなる名詞（形容詞 + 名詞）
- 名詞〜の（名詞 + 接続詞）
- 名詞が動詞（主語 + 動詞）
- 名詞の名詞（所有格）
- 形容詞名詞が動詞（修飾された主語 + 動詞）
```

### 生成パターン（English）
```
- Adjective + Noun
- Noun + Verb
- Noun + Preposition + Noun
- The [Adjective] [Noun] [Verb]
- Noun + Verb + Preposition + Adjective + Noun
```

### 📊 生成された詩の例

**日本語詩:**
```
美しき月が燃える

```

**English Poem:**
```
Sky burns
Fire sings to serene dawn
Moment flows beyond beautiful wind
Heart burns in white fire
```

**デュアル言語:**
```
Japanese:
  風が醒める
  永遠の火が流れる
  宇宙の雪

English:
  Distant snow
  Wave burns beyond eternal water
  Time sings
```

### 🔍 興味深い発見
1. **予期せぬ美しさ**: 「地球〜の」「森の星」のような、文法的に少し変だけど詩的な組み合わせが生まれた
2. **矛盾の美学**: 「美しき月が燃える」「静寂が歌う」のように、矛盾する言葉の組み合わせが詩的になる
3. **言語差**: 日本語は名詞中心の情緒的表現、英語は動詞中心の動的表現になりがち

### 💡 技術的ポイント
- 単語プールのバランス（名詞多め、動詞少なめ）
- パターンのランダム選択で変化をつける
- 単純なルールから複雑な表現が生まれる
