## 実験内容

### アイデア
**「Harmony Hue - 色相環アニメーションアート」**: HSVカラーモデルを使った美しい色相環アニメーションアートを作成。色が滑らかに変化しながら、複数の幾何学パターン（スパイラル、円、ハロー、花びら）が調和して描画される。

### 実装の技術的詳細

#### 使用技術
- **HTML5 Canvas API**: 高性能な2Dグラフィックス描画
- **HSVカラーモデル**: 色相・彩度・明度の直感的な色操作
- **JavaScript**: アニメーションロジックとインタラクション

#### 主要機能
1. **HSV→RGB変換関数**: カラーモデル間の変換を実装
   ```javascript
   function hsvToRgb(h, s, v) { ... }
   ```

2. **複数の描画レイヤー**:
   - `drawSpirals()`: 回転するスパイラルパターン
   - `drawCircles()": 拡大する同心円
   - `drawHalo()`: 分割された円弧（ハロー）
   - `drawPetals()`: 回転する花びらパターン
   - 中心のグロー効果

3. **インタラクティブ性**:
   - マウス位置でアニメーション速度とスケールを変化
   - クリックでベース色相をランダムリセット

4. **トレイル効果**:
   ```javascript
   ctx.fillStyle = 'rgba(10, 10, 15, 0.1)';
   ctx.fillRect(0, 0, width, height);
   ```
   - 前のフレームを完全に消さずに薄く上書きすることで、動きの余韻を表現

### 発見と学び

1. **HSVカラーモデルの美しさ**
   - 色相（Hue）を連続的に変化させるだけで、非常に美しいグラデーションが作れる
   - 彩度と明度を別々に制御できるため、発色の調整が容易

2. **レイヤーの調和**
   - 複数の幾何学パターンを重ねることで、単調になりにくい
   - 異なる回転速度や方向を持たせることで、動きに変化をつけられる

3. **インタラクティブ性の重要性**
   - マウス操作で変化が起きることで、ユーザーが能動的に楽しめる
   - パラメータ制御が視覚的にフィードバックされるのは満足感が高い

4. **トレイル効果の威力**
   - 完全にクリアするのではなく、薄くフェードさせることで動きが滑らかに見える
   - 「動きの余韻」がアート作品としての深みを生む

### 技術的ポイント

- **Canvasのパフォーマンス**: requestAnimationFrameで効率的なアニメーションループ
- **数式の活用**: 三角関数（sin, cos）で円運動や螺旋を表現
- **色の計算**: 色相環（0-360度）を活用した色の遷移
