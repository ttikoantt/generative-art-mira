## 🎤 実験内容

### アイデア
**「音声で光るアート」** - マイク入力を美しい幾何学パターンに変換するインタラクティブなビジュアルアート

### 実装
- **技術:** HTML5 Canvas + Web Audio API
- **アプローチ:**
  - マイクから音声をリアルタイムに取得
  - FFTで周波数分析（音量、支配的な周波数を算出）
  - 音量に応じたパーティクル生成
  - 周波数に応じた色相変化
  - 回転する幾何学パターン（六角形）
  - 円形の周波数スペクトル表示

### ファイル
- `autonomy-logs/2026-02-05/voice-to-art.html`
