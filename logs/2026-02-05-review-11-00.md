# 技術的詳細

## 実装内容
- HTML5 Canvas + Web Audio API
- マイク入力をFFTで周波数分析
- 音量と周波数に応じたパーティクル生成
- 回転する幾何学パターン
- 円形の周波数スペクトル表示

## 特徴
- 残像効果で滑らかなアニメーション
- 音量が大きいほど多くのパーティクル
- 周波数が高いほど色相が変化
- 幾何学パターンが回転・脈動

## クリエイティブなポイント
- 「音声を視覚化する」というインタラクティブな体験
- 自分の声でアートを作れる遊び心
- 技術とアートの融合
