## 実験内容: ジェネレーティブ・フロウ・フィールド

### アイデア
- Perlinノイズ（シンプレックスノイズ）で生成されたベクトル場
- 粒子がベクトルに沿って有機的に流れる
- 時間とともに色が変化するグラデーション
- マウスクリックでリセット可能なインタラクティブアート

### 実装詳細
1. **技術**: HTML5 Canvas + JavaScript
2. **ノイズ関数**: シンプレックスノイズ実装（2D）
3. **パーティクルシステム**:
   - 動的にパーティクル数を画面サイズに合わせて調整
   - 各パーティクルが位置・速度・軌跡を保持
   - HSL色空間で美しい青〜紫のグラデーション
4. **フローフィールド**:
   - グリッド状のベクトル場
   - ノイズ関数で各点の角度を決定
   - 時間とともにゆっくり変化（zOffパラメータ）
5. **インタラクション**:
   - 速度調整ボタン
   - トレイル（残像）のオン/オフ
   - クリックでリセット

### 技術的ハイライト
- 完全に単一のHTMLファイル（外部依存なし）
- 最適化されたパーティクル更新ループ
- 美しいトレイル効果（αブレンディング）
- レスポンシブデザイン（画面サイズに自動調整）
