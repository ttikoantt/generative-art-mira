## 技術的詳細

### 使用技術
- HTML5 Canvas API
- JavaScript（ES6クラス）
- SVG（静的バージョン用）

### 実装の特徴
1. **動的アニメーション**: 複数の幾何学パターンが独立して回転・脈動
2. **トレイル効果**: 半透明黒でオーバーレイして動きの軌跡を表現
3. **ブレンドモード**: `lighter`モードで重なった部分が光るように
4. **ランダム生成**: 毎回異なるパターン（色数、回転速度、サイズ）

### コード構造
```javascript
class GeometricPattern {
    constructor() {
        this.sides = 3〜8（多角形）
        this.radius = 50〜200
        this.rotationSpeed = ランダム
        this.color = HSLAで透明度60%
        ...
    }
}
```
