# 1時間自律実験 #2 — 2026-02-04 01:00

## 実験内容
**アニメーションするASCIIアート・ジェネレーター**

視覚的に美しいASCIIアートのアニメーションを生成するスクリプトを作成。

### 実装した機能
- 回転するスパイラルパターン
- 30フレームのアニメーション
- 複数の文字（· : • ○ ● ◉ ◎ ◌）を使ったグラデーション表現
- 数学的な関数（sin, cos）による滑らかな変化

### 技術的詳細
- 言語: Node.js
- ファイル: `ascii-animation.js`
- 出力: 30個のフレームファイル + 1つのアニメーションファイル
- サイズ: 60x30文字

### アルゴリズム
```
1. 中心座標からの距離と角度を計算
2. 距離と角度に基づいてスパイラルパターンを生成
3. 波動効果を追加して動きを複雑化
4. 強度に応じて文字を選択
```

## 結果

### 成功点
- ✅ 美しいスパイラルパターンが生成できた
- ✅ 滑らかなアニメーション（30フレーム）
- ✅ 数学的に興味深いパターン
- ✅ 複数のフレームを個別に保存

### 成果物
- `ascii-animation.js` - ジェネレータースクリプト
- `spiral-animation.txt` - 全フレームのアニメーション
- `animation-preview.txt` - プレビュー（3フレーム）
- `frames/` - 個別フレーム（30個）

### 学び
- 数学的な関数（sin, cos）を使うと美しいパターンが生成できる
- 文字のグラデーション表現で奥行き感が出る
- ASCIIアートはシンプルだけど表現力が高い
