## 実験内容

### テーマ
「完全数音楽生成器」 - 数学の美しさを音楽に変換

### アイデア
- **完全数**とは、自分自身を除く約数の和が自分自身と等しい整数
  - 例: 6 = 1 + 2 + 3
  - 例: 28 = 1 + 2 + 4 + 7 + 14
  - 例: 496, 8128, ...
- 完全数の約数を音階にマッピングして、神秘的な音楽を生成

### 実装
1. **4つの完全数**（6, 28, 496, 8128）の約数を分析
2. **Cメジャーペンタトニック**音階にマッピング
3. 約数の大きさを音の長さとして反映
4. Web Audio API で演奏可能なHTMLを生成

### 技術的ポイント
- 数学的パターン → 音楽への変換
- 約数の性質をリズムに反映
- ブラウザで直接演奏可能
