## 実験内容
**インタラクティブ・リサージュ図形アートジェネレーター**

リサージュ曲線（Lissajous curves）は、2つの垂直方向の単振動を合成して得られる図形です。シンプルな三角関数から驚くほど複雑で美しい幾何学模様が生まれます。

### 実装の特徴

**技術的要素:**
- Canvas API でのリアルタイム描画
- パラメータ制御（周波数X/Y、位相シフト、速度、トレイル）
- グラデーションストロークとグロー効果
- アニメーションフェード効果

**インタラクティビティ:**
- 5つのスライダーでリアルタイム調整
- Rキー：ランダムプリセット生成
- スペースキー：一時停止/再生
- 現在の数式と周波数比を表示

**視覚的デザイン:**
- ダークテーマ (#0a0a0f)
- 紫・マゼンタ・ピンク・青・シアンのグラデーション
- ガラスモーフィズム風コントロールパネル
- トレイル効果で動きの軌跡を美しく表現

### 数学的背景

リサージュ曲線のパラメトリック方程式:
```
x = A·sin(at + δ)
y = B·sin(bt)
```

- a, b: 周波数（図形の複雑さを決定）
- δ: 位相シフト（図形の回転・変形）
- 周波数比が有理数の場合、閉じた曲線になる

### 発見・学び

**美的発見:**
1. **シンプルな法則から複雑な美が生まれる**: sin と cos の組み合わせだけで、無限のバリエーション
2. **整数比の神秘**: 3:2, 5:4, 7:6 などの簡単な整数比が特に美しい図形を作る
3. **位相の魔法**: π/2 の位相シフトが図形を対称的に美しくする

**技術的学び:**
1. **Canvasグラデーション**: createLinearGradient で動的な色遷移を実現
2. **トレイル効果**: 半透明の黒で上書きすることで、動きの軌跡を残す
3. **数学的可視化**: 数式を視覚的に理解できる良い教材

**面白いプリセット:**
- 3:2, π/2 → 無限ループ風
- 5:4, π/2 → 複雑な結び目
- 7:6, π/2 → 幾何学的花模様
- 9:8, π/2 → 螺旋的な美しさ
