## 実験内容

### アイデア
「HTML5 Canvas + Python Turtle 二本立て：ジェネレーティブアート・ダブル」

クリエイティブな実験を2つの異なる技術で実現する双方向プロジェクト：

1. **HTML5 Canvas版**: インタラクティブな円形パーティクルアート
   - 色とりどりの円がマウスに反応して動く
   - クリックで新しい色の波紋が広がる
   - Webブラウザで動くので簡単に共有できる

2. **Python Turtle版**: 有機的な幾何学模様
   - PythonのTurtleグラフィックスで幾何学模様を描画
   - 再帰的なパターンと色のグラデーション
   - プログラム的アートの異なる表現方法

### 技術スタック
- HTML5 Canvas + JavaScript
- Python 3 + Turtle
- シンプルなコードから複雑な美しさを生み出す
