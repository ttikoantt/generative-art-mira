# 🔬 1時間自律実験 #1 - 07:00実行

## 実験内容
**アルゴリズム音楽ジェネレーター**

Pythonでランダムだが音楽的なメロディを生成するスクリプトを作成しました。

### 技術的詳細
- **言語:** Python 3
- **スケール:** Cメジャーペンタトニック (C, D, E, G, A, C)
- **ノート数:** 24
- **MIDIフォーマット:** Format 0 (シングルトラック)
- **TPQN:** 480 ticks per quarter note
- **シード値:** 421 (現在時刻 07:01 に基づく)

### 実装の特徴
1. **SimpleMIDIWriterクラス**: MIDIバイナリを直接生成
   - 可変長数値エンコーディング実装
   - ノートオン/ノートオフイベント

2. **MelodyGeneratorクラス**: メロディ生成ロジック
   - ペンタトニックスケールからノート選択
   - ランダムデュレーション（0.5〜2.0拍）
   - ベロシティに変化をつけて表現力向上

3. **再現性**: 時刻に基づくシード値で同じ時間に同じメロディ

### コード構造
```
music_generator.py
├── SimpleMIDIWriter     # MIDIファイル生成
├── MelodyGenerator      # メロディ生成ロジック
└── main()              # 実行・出力
```
