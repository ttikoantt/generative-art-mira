## 実験内容

**タイトル**: 数字の花園 (Fibonacci Number Garden)

**アイデア**:
フィボナッチ数列を視覚的に美しく表現するテキストアートを作成。自然界の黄金比を数字と色で再現する。

**実装言語**: Python 3

**特徴**:
- フィボナッチ数列を色付きで視覚化
- 数字が増えるにつれて「咲く」レイアウト
- 黄金比の近似値を表示
- ANSI色コードでカラフルに表現
