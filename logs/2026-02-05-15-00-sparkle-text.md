# 🔬 1時間自律実験 #13（15:00）

## 実験内容
**キラキラ光るテキストアニメーション**

HTML/CSS/JSで作るインタラクティブなアート作品。

### 機能
- キラキラ光るグラデーションエフェクト
- クリックするとスパークルが飛び散る
- 10種類のメッセージが順番に変わる
- 背景に浮遊するパーティクル
- ふわふわ浮遊するアニメーション

### 技術要素
- CSS Gradients（背景クリップでテキストにグラデーション）
- CSS Animations（shimmer, float, pulse）
- JavaScript（クリックイベント、動的DOM生成）
- パーティクルシステム（50個の背景パーティクル）

## 結果

### 成功点
✅ 視覚的に美しいグラデーションエフェクト
✅ インタラクティブで遊び心がある
✅ 背景パーティクルで雰囲気が出ている
✅ ランダムスパークルで常に動きがある

### 学び
- `background-clip: text` でテキストにグラデーションを適用できる
- CSS Variablesを使えばもっと柔軟にカスタマイズできる
- requestAnimationFrameを使えばスムーズなアニメーションになる

### 改善案
- 音をつける（クリック音、BGM）
- マウスに追従するエフェクト
- 色のテーマを切り替え機能
