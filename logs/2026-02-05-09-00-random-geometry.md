## コンセプト
**「ランダム・ジオメトリ・ジェネレーター」**

HTML5 Canvasを使ったジェネレーティブ・アート。10種類の調和カラーパレットと5種類の図形タイプを組み合わせて、無限のバリエーションのアートを生成するWebアプリケーション。

---

## 技術スタック
- HTML5 Canvas API
- Vanilla JavaScript
- CSS Animations
- 数学関数（三角関数、乱数）

---

## 機能詳細

### カラーパレット（10種類）
1. **Sunset vibes** - 夕暮れの温かみ（#ff6b6b, #feca57, #ff9ff3...）
2. **Ocean depths** - 海の深淵（#0abde3, #48dbfb, #00d2d3...）
3. **Forest magic** - 森の魔法（#10ac84, #1dd1a1, #00d2d3...）
4. **Neon dreams** - ネオンの夢（#fd79a8, #e84393, #a29bfe...）
5. **Warm embrace** - 暖かい抱擁（#fdcb6e, #e17055, #d63031...）
6. **Cool elegance** - クールな優雅さ（#74b9ff, #0984e3, #a29bfe...）
7. **Earth tones** - 大地の色（#b7950b, #7d6608, #9a7d0a...）
8. **Berry blend** - ベリーのブレンド（#e056fd, #686de0, #30336b...）
9. **Pastel dreams** - パステルの夢（#ffcccc, #ccb3ff, #b3d9ff...）
10. **Cyberpunk** - サイバーパンク（#00ff87, #60efff, #ff00ff...）

### 図形タイプ（5種類）
- **円（Circle）**: 最も基本的で美しい図形
- **矩形（Rectangle）**: 直線的な構造を作る
- **三角形（Triangle）: 動きと方向性を表現
- **多角形（Polygon）**: 5-8辺の複雑な形
- **ライン（Line）**: 直線的なアクセント

### ブレンドモード（12種類）
source-over, multiply, screen, overlay, darken, lighten, color-dodge, color-burn, hard-light, soft-light, difference, exclusion

### アニメーション機能
- 各シェイプ個別のタイミング（delay: 0-1秒）
- Easing関数（ease out cubic）で滑らかな登場
- 20-40個のランダムシェイプが順番に現れる

### UI機能
- **Generateボタン**: 無限に新しいアートを生成
- **Saveボタン**: 高解像度PNGで保存
- **パレットプレビュー**: 使用中の色を表示

## アルゴリズム

### 生成プロセス
1. ランダムにカラーパレットを選択
2. 20-40個のシェイプオブジェクトを生成
   - 位置、サイズ、色、透明度、回転、ブレンドモードをランダム設定
3. 各シェイプに個別のアニメーションタイミングを付与
4. requestAnimationFrameでアニメーション開始
5. スムーズなeasingで徐々に登場

### 乱数の活用
- 均一分布で図形タイプを選択
- 三角関数で多角形の頂点を計算
- 透明度とサイズで深度を表現
