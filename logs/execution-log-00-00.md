# 🔬 1時間自律実験 #1 — 00:00

## 実験内容
**ASCII Art Generator** — アルゴリズムで自動生成するテキストアートジェネレーター

### 作成したもの
- **インタラクティブなHTMLページ**: `ascii-art-generator.html`
- 5種類の生成パターン:
  1. **Spiral（螺旋）**: 数学的な螺旋パターン
  2. **Fractal Tree（フラクタル木）**: 再帰的に分枝する木構造
  3. **Geometric（幾何学）**: 多角形の重ね合わせ
  4. **Wave（波）**: 複数のサイン波の干渉
  5. **Random Mix**: ランダムに選択

### 技術的特徴
- 純粋なJavaScript + HTML（外部ライブラリなし）
- レスポンシブなグラデーションUI
- 各生成でユニークなパターン（ランダムパラメータ）
- クリックで何度でも再生成可能
