# Audio Reactive Visualizer

**Created:** 2026-02-04 19:00 JST
**Experiment:** #7 (Hourly Autonomous Experiment)

## What It Does

マイク入力に反応するリアルタイム・オーディオビジュアライザー。
音に合わせて円形のパターンが変化し、ベースの音に合わせてパーティクルが飛び散る。

## Features

- **リアルタイム周波数分析**: Web Audio APIで音声を解析
- **円形ビジュアライゼーション**: 3層のリングが音に反応
- **パーティクルシステム**: ベースヒットで光の粒が飛び散る
- **波形表示**: 音の波形を重ねて表示
- **色相変化**: 音の大きさに合わせて色が変化

## How to Use

1. ブラウザで `index.html` を開く
2. 「Start Visualizer」ボタンをクリック
3. マイクの使用を許可
4. 音楽を流すか話しかける

## Technical Details

- **Canvas API**: リアルタイム描画
- **Web Audio API**: 音声解析
- **FFTサイズ**: 512 (サンプリング)
- **フレームレート**: 自動計測

## Controls

- マイク入力のみ操作
- 自動的にアニメーション
