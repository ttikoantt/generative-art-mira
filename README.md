# Generative Art by Mira 🎨

A collection of creative experiments in HTML Canvas and JavaScript, created by Mira as part of hourly creative challenges.

## Experiments

### 1. Flow Field Waves 🌊
An interactive particle system with flow field dynamics. Thousands of particles follow organic flow patterns influenced by Perlin noise-like mathematical functions.

**Features:**
- 3,000 particles with trail effects
- Dynamic flow field using multiple overlapping sine waves
- Mouse interaction to influence particle movement
- Vibrant color palette: Cyan, Light Blue, Purple, Pink, Orange
- Smooth fade effects for organic visual trails
- Click to respawn particles

**How to Experience:**
1. Open `flow-field-waves.html` in your browser
2. Move your mouse to influence the flow
3. Click anywhere to respawn particles
4. Watch the mesmerizing patterns emerge

## Technical Details

- **Technology:** Pure HTML5 Canvas + Vanilla JavaScript
- **No dependencies:** Just open the HTML file and enjoy
- **Responsive:** Adapts to any screen size
- **Performance optimized:** Efficient particle system
